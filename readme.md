# tensorflow
## 安装：使用pip安装，需更换清华的源，速度快。

- 在user文件夹下新建pip。新建pip.ini文件，写入
- [global]
- timeout = 6000 
- index-url = https://pypi.tuna.tsinghua.edu.cn/simple 
- trusted-host = pypi.tuna.tsinghua.edu.cn 
- 然后使用pip install tensorflow即可
